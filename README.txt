Execução do programa em Windows (necessário Netbeans):

- Descompacte o .zip

- No Netbeans, abra o projeto "Biblioteca" que está na pasta onde o .zip foi descompactado

- Aperte F6 para para executar o projeto

- Selecione uma das seguintes opções:

	1)Cadastrar usuarios: cadastra um usuário com RG, NOME e Tipo de usuario (aluno, professor e comunidade) clicando em cadastrar para finalizar o cadastro ou em voltar para cancelar

	2)Registrar empréstimo: registra o empréstimo através do ID do livro, RG do usuario e data do empréstimo (dd/mm/yyyy) clicando em emprestar para finalizar o empréstimo ou em voltar para cancelar

	3)Deletar empréstimo:

	4)Cadastrar livros: cadastra um livro com ID, Nome e tipo do livro (texto ou geral) clicando em cadastrar para finalizar o cadastro ou em voltar para cancelar

	5)Listagem de dados: lista os empréstimos já cadastrados no terminal do NetBeans

obs: caso as funções de cadastrar e voltar não estejam visíveis, clicar nas bordas e arrastar para aumentar a janela!

- Clique em fechar para encerrar o programa
